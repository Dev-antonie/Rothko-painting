# Rothko Painting

A Rothko-inspired painting built with HTML and CSS.

## About

This project recreates the style of Mark Rothko's abstract expressionist paintings using only HTML and CSS. It focuses on soft-edged color fields, warm muted tones, and the atmospheric quality of Rothko's work.

## Built With

- HTML
- CSS

## What I Learned

- CSS box model and layout
- filter and blur effects
- linear gradients
- box-shadow for soft edges
- Git and GitHub workflow
